# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2025-07-19

### Added
- Premium contact/demo request form with validation, accessibility, and loading/success/error states.
- Comprehensive SEO metadata: Open Graph, Twitter cards, canonical URL, theme-color, web manifest, and multi-size icons.
- JSON-LD structured data for Organization and SoftwareApplication.
- SVG favicon (`app/icon.svg`) and web app manifest (`public/manifest.webmanifest`).
- Money-back guarantee trust badges in the pricing section.
- Feature comparison table with icon-based availability indicators.
- Pricing-specific FAQ accordion.
- Enterprise CTA block with gradient background.
- Professional README, LICENSE, CHANGELOG, and DOCUMENTATION files.

### Changed
- Redesigned pricing cards with monthly/yearly toggle and animated price transitions.
- Professional plan visually highlighted with premium border, soft glow, and "Most Popular" badge.
- Footer links updated with Privacy Policy, Terms of Service, Cookie Policy, Documentation, Support, and Status.
- Social icons now point to realistic profile URLs.
- Package metadata updated for commercial distribution (name, author, description, license).

### Removed
- Unused `motion` import from `components/sections/case-study.tsx`.

## [1.0.0] - 2025-07-18

### Added
- Initial release of the Agentix AI landing page template.
- Hero, social proof, features, how-it-works, industries, case study, testimonials, pricing, FAQ, final CTA, and footer sections.
- Dark/light theme with system-aware toggle and no-flash boot script.
- Responsive mobile-first layout from 320px to ultrawide.
- Framer Motion animations with `prefers-reduced-motion` support.
- shadcn/ui component library pre-installed.
